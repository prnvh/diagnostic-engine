Structured knee symptom triage, packaged for a Vercel demo.

This demo takes a free-text complaint, converts it into controlled evidence, asks a few discriminating questions, and returns ranked fit scores or safety escalation. It is not a diagnosis.

  • State survives across stateless serverless invocations.
  • Every session keeps an append-only audit ledger.
